Voilà notre projet, on a fait tous les exercices comme démandé, sans toutefois créer une classe couleur pour le P14 (voir le fichier REPONSES pour plus de détails). On a aussi fait des montagnes aléatoires et des rivières.

Pour compiler notre projet, taper "make" depuis le terminal dans le dossier projet.

Taper ensuite ./Application dans le dossier projet pour lançer la simulation 3D.

Pour l'utilisation du projet plus pousée voir le fichier UTILISATION_DU_PARSEUR.

Le dossier code contient tous les .h et .cc .

Dans notre projet, on vous conseille de regarder les rivières et les montagnes aléatoires, car il s'agit d'extensions du projet de base, et le parseur il est très efficace.
